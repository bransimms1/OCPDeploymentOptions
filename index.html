What is NGIDE?

NGIDE is a unified, automated, and integrated software delivery pipeline and runtime environment. It is built on five core principles designed to accelerate innovation and streamline development.

Automation

Streamline development and operational tasks.

Consistency

Ensure uniform environments across the lifecycle.

Scalability

Adapt to changing workload demands with ease.

Security

Integrate security throughout the development process.

Self-Service

Empower developers with on-demand resources.

Deconstructing the NGIDE Architecture

The platform is composed of two distinct, cooperative layers that provide a complete solution for development and operations.

Red Hat OpenShift (The Orchestrator)

Built on Kubernetes, OpenShift manages applications and provides a full suite of developer tools, CI/CD, security, and operational management.

Infrastructure (The Foundation)

The physical or virtualized hardware providing compute, memory, storage, and networking. This can be hosted in the cloud or on-premises.

Understanding Red Hat OpenShift Subscriptions

The right subscription depends on your chosen infrastructure and workload needs. OpenShift provides tailored models for maximum performance and flexibility.

OCP (Core-Based)

Designed for virtualized infrastructure like AWS EC2. Licensing is calculated based on the number of virtual CPU cores used by worker nodes.

Best for: Cloud Deployments

OCP (Bare Metal)

Intended for direct installation on physical servers. This model provides optimal performance and is licensed per physical CPU socket or core.

Best for: On-Premises Performance

OpenShift Virtualization

An add-on for OCP on bare metal. It enables running traditional Virtual Machines (VMs), including Windows, alongside containers on the same cluster.

Key Differentiator for On-Premises
